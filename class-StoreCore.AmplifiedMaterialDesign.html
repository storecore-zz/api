# minify( boolean $minify = true )

Extreme minification.

Extreme minification.

By default the parent method MaterialDesignLite::minify() uses readable shorthand replacements for MDL CSS class names. This method adds an additional minification by using even shorter class names from ma to mz and then maa to mzz.
