# validate( mixed $variable )

Validate a variable.

Validate a variable.

By default, all StoreCore data types are internally validated using Standard PHP Library (SPL) exceptions. The additional static method validate() provided by this interface may be used as a shortcut for validating variables, constants, and input values. This method acts as a helper function that eliminates the need to instantiate a new object and handle exceptions.

Parameters

$variable
The variable, constant, or value to validate.

Returns

boolean

Returns true if a variable matches a given StoreCore type or false if it does not.

Throws


The validate() method MUST NOT throw any exceptions, but merely return a boolean true or false.
