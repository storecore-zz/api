public static string|false
# save( string|array $tables = '*', boolean $drop_tables = false )

Save a database backup to file.

Save a database backup to file.

Parameters

$tables

Optional comma-separated string or array with the names of database tables to save. Defaults to '*' for all tables.

$drop_tables

If set to true, DROP TABLE <tbl_name> clauses are added to delete existing database tables. Defaults to false, so only non-existent database tables are recreated with the CREATE TABLE IF NOT EXISTS clause.

Returns

string|false

Returns the filename of the SQL backup file on success or false on failure.
