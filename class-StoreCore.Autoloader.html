Class Autoloader

PSR-4 Compliant Framework Autoloader

This autoloading class links a PHP namespace to a directory or multiple directories. Linking a 'Foo\Bar' namespace to its base directories looks something like: $loader->addNamespace('Foo\Bar', '/path/to/packages/foo-bar/src'); $loader->addNamespace('Foo\Bar', '/path/to/packages/foo-bar/tests');
